# JupyterLite: run all your code in a web browser using WebAssembly

- PyData London 2026
- 2026-06-06 11:50

## Build locally

You need to have `node.js` installed.

To build the HTML slides use

```bash
npm install
npm run html
```

which generates `output/index.html`

To generate a PDF file of the slides use

```bash
npm run pdf
```

which generates `jupyterlite.pdf`.
